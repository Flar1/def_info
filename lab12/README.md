# Ментальный покер - Техасский холдем

## Описание

Реализация алгоритма "Ментальный покер" с графическим интерфейсом для игры в Техасский холдем. Система позволяет играть в покер без доверенного третьего лица, используя криптографические методы для обеспечения безопасности и честности игры.

## Особенности

- Поддержка произвольного количества игроков (от 2 до 10)
- Графический интерфейс на основе Tkinter
- Криптографическая защита на основе RSA
- Полная реализация правил Техасского холдема
- Раздача 2 карт каждому игроку и 5 общих карт на стол

## Установка

### Требования

- Python 3.7 или выше
- Библиотека pycryptodome
- Для GUI версии: tkinter (обычно устанавливается отдельно)

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Установка tkinter (для GUI версии)

**Ubuntu/Debian:**
```bash
sudo apt-get install python3-tk
```

**Fedora/RHEL:**
```bash
sudo dnf install python3-tkinter
```

**Arch Linux:**
```bash
sudo pacman -S tk
```

## Запуск

### Графическая версия (требует tkinter):

```bash
python3 gui.py
```

### Консольная версия (работает без tkinter):

```bash
python3 console_game.py
```

## Использование

1. Выберите количество игроков (от 2 до 10)
2. Нажмите "Начать игру" - система создаст и зашифрует колоду
3. Нажмите "Раздать карты" - каждому игроку будет раздано по 2 карты
4. Используйте кнопки "Флоп", "Терн", "Ривер" для раздачи общих карт
5. Выберите игрока для просмотра его карт в выпадающем списке
6. Используйте кнопки для ставок и других действий

## Структура проекта

- `crypto_mental_poker.py` - Криптографический модуль для шифрования карт
- `mental_poker_game.py` - Логика игры в ментальный покер
- `gui.py` - Графический интерфейс (требует tkinter)
- `console_game.py` - Консольная версия (работает без tkinter)
- `test_mental_poker.py` - Тестовый скрипт
- `SECURITY.md` - Обоснование безопасности и честности схемы
